Contexte du projet

Le projet E-Com est un site e-commerce que j'ai développé en groupe de 3 dans le cadre de ma formation Bachelor Déveleppement Web. Il avait pour objectif de simuler un site de vente en ligne (Motos, accessoires et piéces détachés) avec des fonctionnalités essentielles d’un vrai site marchand. Ce projet m’a permis d’appréhender l’ensemble du cycle de développement web, du front-end au back-end.

Galerie du Projet

Fonctionnalités principales

  • Catalogue produits : affichage dynamique avec images, descriptions, catégories, tri et filtres
  • Panier utilisateur : ajout, modification et suppression d’articles avec gestion de la quantité
  • Tunnel de commande : formulaire de livraison et récapitulatif avant validation
  • Comptes utilisateurs : système de connexion/inscription avec historique des commandes
  • Espace administrateur : CRUD complet des produits et gestion des utilisateurs

Technologies utilisées

  • Framework : Laravel 11
  • Front-end : Blade, Tailwind CSS, JavaScript
  • Base de données : MySQL avec Eloquent ORM
  • Authentification : Laravel Breeze
  • Architecture : MVC (Modèle - Vue - Contrôleur)
  • Outils : Git, GitHub, PHPMyAdmin

Défis rencontrés

Ce projet m’a permis de progresser sur plusieurs aspects :

  • Gestion des relations entre modèles : produits, utilisateurs, commandes, etc.
  • Validation des formulaires : via les règles de validation Laravel et gestion des erreurs
  • Interface Blade dynamique : avec réutilisation de composants et mise en page responsive
  • Sécurité : protection CSRF, hachage des mots de passe, middleware pour les routes
